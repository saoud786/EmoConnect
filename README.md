# EmoConnect

EmoConnect is a real-time chat web application focused on anonymous communication and mental well-being.  
It allows users to connect, chat, and share thoughts securely using a simple and clean interface.

---

## Features

- **Random Anonymous Chat**  
  Connect with new people instantly and chat without revealing identity.

- **Normal Chat System**  
  Search users, send requests, and chat like a standard messaging app.

- **Peace Hub**  
  A calm space with basic wellness features like meditation and relaxation.

- **AI Assistance**  
  Simple AI-based responses for emotional support.

- **Abuse Detection**  
  Messages are monitored for abusive words and flagged automatically.

- **Admin Dashboard**  
  Admin can review flagged messages and block or unblock users.

- **Secure Authentication**  
  Firebase-based login system with email and password.

- **Anonymous Identity**  
  Users are assigned random usernames for privacy.

- **Real-time Updates**  
  Instant messaging powered by Firebase Firestore.

---

- React.js  
- Firebase (Authentication + Firestore)
